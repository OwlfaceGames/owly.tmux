## Installation

### Using TPM (Recommended)

1. Add plugin to the list of TPM plugins in `~/.tmux.conf`:
   ```bash
   set -g @plugin 'OwlfaceGames/owly.tmux'
   ```

2. Hit `prefix + I` to fetch and source the plugin.
